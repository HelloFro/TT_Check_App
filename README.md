# T-T Check App

CST407 - Mobile App Developement in Android

Developer: Fro Warner

## Overview

App that implements Tic-Tac-Chess, but with a 3x3 grid and different rules.

## Features

- Animated Widgets (2pt)
- Shared Preferences (1pt)
- Fragment (2pt)
- Async Tasks (1pt)
- Menu Items (1pt)
- List View (1pt)
- Grid View

## Notes

### Omitted Proposal Points

- External Api (2pt)
- Firebase (2pt)
- > 3 Activities (1pt)
